
# 🚀 Modern Developer Portfolio

A premium, responsive developer portfolio inspired by modern design trends (similar to arshin.me), built with **HTML, CSS, and JavaScript**.

This project showcases real deployed applications with interactive UI, smooth animations, and a clean, professional layout.

## 📌 Features

- ✨ Modern dark UI design
- 🎯 Smooth scroll animations (Intersection Observer)
- 🖱️ Hover effects and interactive project cards
- 📱 Fully responsive (mobile + desktop)
- 🔗 Clickable project previews (live demos)
- ⚡ Fast and lightweight (no frameworks)
